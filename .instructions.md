# Repository AI Instructions

High-level instructions for AI agents editing this repository.

- Respect `.AI-GUIDELINES.md` and follow the `pre-commit` rules in `.pre-commit-config.yaml`.
- Keep edits small. Include rationale for non-trivial changes in commit messages.
- Do not modify files under `.secrets/` or attempt to populate secret placeholders.
- When adding new executable scripts, ensure they include a `set -euo pipefail` header and a short usage message.
