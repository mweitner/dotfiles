# Copilot / AI Agent Instructions for dotfiles

Purpose: Provide per-repo guidance for Copilot and other AI agents.

- Preserve sensitive data; never create or embed secrets, credentials, or private keys.
- Prefer conservative edits and open small PRs for review.
- Run pre-commit locally and ensure `shellcheck` / `shfmt` and `markdownlint` pass before proposing changes.
- For installer or system-altering scripts, include a verification checklist in the commit body.
