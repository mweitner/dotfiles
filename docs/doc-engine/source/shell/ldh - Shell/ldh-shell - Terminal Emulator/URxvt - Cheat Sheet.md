# URxvt - Cheat Sheet

# General

| Command | Description |     |
|---------|-------------|-----|
| `Ctrl+g` | clear terminal |     |
| `Ctrl+Alt+v` | paste clipboard to terminal |     |
| `Ctrl+<right_mouse>, <right_mouse> while selecting` | copy over a selection in rxvt, and a contextual menu appears with options for copying and pasting. Once the menu appears, you may release the **Ctrl** key, but you must keep the right mouse button pressed until you make your selection |     |

# URxvt's Tabs Plugin

Shared and installed by dotfiles repo and symlinked at `$XDG_CONFIG_HOME/X11/.Xresources`

CTRL+SHIFT+N - open a new tab

CTRL+SHIFT+K - select next tab

CTRL+SHIFT+J - select previous tab

CTRL+SHIFT+R - rename tab

CTRL+SHIFT+H - move tab to the left

CTRL+SHIFT+L - move tab to the right

CTRL+d - close tab (close pseudo terminal PTY of current tab)
